IP = Internet Protocol. huolehtii viestin kuljetuksesta "päästä päähän".

Jokaisella laitteella on oma IP osoite. Se voi olla globaali tai paikallinen.

TCP = Transmission Control Protocol.

tiedonsiirto protokolla. Hallittavissa. Avataan portti kahden pisteen välillä ennen kuin viesti voidaan lähettää

UDP = tiedonsiirto protokolla.

DNS = Domain Name System.

IP osotteiden muistaminen hankalaa. DNS on tekstimuotoinen nimi. Nimipalvelimet huolehtii domainnimien muuttamisesta IP osoiteiksi.

URL = Uniform Resource Locator

Osoittaa haettavan kohteen.

HTTP Protokolla

Virhekoodit

  • 1xx - vastaanotettu, käsittely jatkuu
  • 2xx - kkäsitelty onnistuneesti
  • 3xx - uudelleenohjaus - kohde muuttanut, tarvitaan uusikutsu
  • 4xx - kutsusta johtuva virhe
  • 5xx - palvelimella tapahtui virhe

HTTP metodeita

  • GET - resurssien hakua varten (esim. verkkosivun hakuun)
  • POST - täytetyn lomakkeen tietojen lähetys palveluun
